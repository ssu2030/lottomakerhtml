### lotto number maker

- 개요: 마크다운 언어에 javascipt를 넣어 동작 시키기 위해 만들어본 예제
- 학습내용: css 스타일링 방법(class, id, inline style), javascipt 코드 function 작성 후 이벤트 바인딩, 웹폰트 입히는 법 학습
